API Documentation

Complete reference for the NodeWeaver RAG Classifier API

API Overview
Base URL
https://your-domain.com/api/v1
Content Type
application/json
Response Format

All responses are in JSON format

Rate Limiting

Currently no rate limiting implemented

Classification Endpoints
POST /classify

Classify a single text input

Request Body:
{
  "text": "Schedule dentist appointment for next Tuesday",
  "metadata": {
    "source": "user_input",
    "priority": "medium"
  }
}
Response:
{
  "predicted_category": "personal",
  "confidence_score": 0.85,
  "similar_topics": [
    {
      "topic_id": 1,
      "label": "Personal: Health appointments",
      "category": "personal",
      "similarity": 0.92
    }
  ],
  "similar_nodes": [
    {
      "node_id": 15,
      "content": "Book doctor appointment",
      "category": "personal",
      "similarity": 0.88
    }
  ],
  "document_id": 123,
  "processing_time": 0.145
}

Topic Management Endpoints
GET /topics

Get all topics with optional filtering and pagination

Query Parameters:
  • category - Filter by category
  • min_weight - Minimum topic weight
  • min_coherence - Minimum coherence score
  • page - Page number (default: 1)
  • per_page - Items per page (default: 20, max: 100)
Example Request:
GET /api/v1/topics?category=work&min_weight=2.0&page=1&per_page=10
POST /topics/detect

Trigger topic detection algorithm to find emerging topics

Response:
{
  "message": "Topic detection completed",
  "emerging_topics": 3,
  "topics": [
    {
      "topic_id": 25,
      "label": "Work: Project management tasks",
      "category": "work",
      "total_weight": 4.2,
      "coherence_score": 0.78,
      "origin_node_ids": [45, 67, 89, 102]
    }
  ]
}

Find topics similar to input text

Request Body:

Integration Examples
JavaScript
// Classify text
async function classifyText(text) {
  const response = await fetch('/api/v1/classify', {
    method: 'POST',
    headers: {
      'Content-Type': 'application/json',
    },
    body: JSON.stringify({ text: text })
  });
  
  const result = await response.json();
  return result.predicted_category;
}
Python
import requests

def classify_text(text):
    response = requests.post(
        'https://your-domain.com/api/v1/classify',
        json={'text': text}
    )
    
    result = response.json()
    return result['predicted_category']

# Example usage
category = classify_text("Schedule team meeting")
print(f"Category: {category}")
cURL
curl -X POST "https://your-domain.com/api/v1/classify" \
  -H "Content-Type: application/json" \
  -d '{
    "text": "Buy groceries and cook dinner",
    "metadata": {"source": "todo_app"}
  }'
Google Apps Script
function classifyTask(text) {
  const response = UrlFetchApp.fetch(
    'https://your-domain.com/api/v1/classify',
    {
      method: 'POST',
      headers: {
        'Content-Type': 'application/json'
      },
      payload: JSON.stringify({text: text})
    }
  );
  
  const result = JSON.parse(response.getContentText());
  return result.predicted_category;
}
